# Code of Conduct

## Our Commitment

The Pollution Control Hub community is dedicated to providing a welcoming, respectful, and inclusive environment for everyone who contributes, participates, or interacts with the project.

We value collaboration, transparency, and constructive discussions to build tools that help communities understand and address environmental challenges.

## Expected Behavior

Community members are expected to:

- Be respectful and considerate toward others.
- Provide constructive feedback during discussions and code reviews.
- Respect different opinions and approaches.
- Help maintain a positive and collaborative environment.
- Submit accurate and responsible pollution reports when using community features.
- Respect the privacy and safety of other users.

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, discrimination, or personal attacks.
- Trolling, intentionally disruptive behavior, or disrespectful communication.
- Publishing private information without permission.
- Submitting false or misleading reports intentionally.
- Any behavior that negatively impacts the safety and experience of the community.

## Contribution Guidelines

When contributing code, documentation, issues, or suggestions:

- Explain changes clearly.
- Respect maintainers' decisions and review feedback.
- Focus discussions on improving the project.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Actions may include removing inappropriate content, restricting participation, or other measures depending on the situation.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it by opening an issue in the repository or contacting the project maintainers.

Thank you for helping make Pollution Control Hub a respectful and collaborative community.
